## 🚀 クイックスタート

```bash
# リポジトリのクローン
git clone https://github.com/foresthill/quantum-neural-website.git
cd quantum-neural-website

# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ブラウザで開く
# http://localhost:5173
```
